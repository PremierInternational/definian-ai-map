# Definian AI Integration Map

An interactive registry of all AI agents and assistants integrated into the Definian Data Platform.

## Features
- Expandable/collapsible agent hierarchy
- Status tracking (Completed / In Progress / Not Started)
- Filter by status, owner, or build type
- Add, edit, and delete agents directly in the UI
- Export to standalone HTML

## Viewing
Open `index.html` in any browser — no setup required.

## Updating
1. Open the live app or `index.html` locally
2. Make edits using the UI
3. Click **⬇ Export HTML** to download the updated file
4. Replace `index.html` in this repo and push

## Tech Stack
- React 18 (via CDN, no build step)
- Titillium Web + Roboto (Google Fonts)
- Zero dependencies beyond a browser
